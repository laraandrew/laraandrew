## Work

**RL / AI Research**

- [`ToolOrchestratorEnv`](https://github.com/laraandrew/ToolOrchestratorEnv) — OpenEnv-compatible RL environment where an agent learns cost-aware tool-selection policy across HotpotQA, MATH, GPQA, and HumanEval. Extends SearchEconomicsEnv to a 6-tool action space with a shared cost budget.
- [`reasoningeconomicsenv`](https://github.com/laraandrew/reasoningeconomicsenv) — RL environment for reasoning compute allocation. The agent learns how many tokens to spend per question across a fixed-budget episode rather than solving math directly.

**Applied Projects**

- [`Gesture-LL`](https://github.com/laraandrew/Gesture-LL) — FastAPI backend for a gesture- and voice-driven language learning loop. MediaPipe gesture pipeline, async WebSocket state broadcast, Whisper-ready STT abstraction.
- [`GoAdventure`](https://github.com/laraandrew/GoAdventure) — Expo / React Native trip planning app with weather-aware guidance. Runs on iOS, Android, and web.
- [`Nextcloud-Cookbook-iOS`](https://github.com/laraandrew/Nextcloud-Cookbook-iOS) — Native iOS/iPadOS/macOS client for Nextcloud Cookbook. Swift/SwiftUI, extended with settings lifecycle hardening and date parsing fixes.

- [`Portfolio-Website`](https://andrew-lara.com) — Personal site built with React, TypeScript, and Tailwind CSS.
